Kilotest: Test recommendations

How recommendations work

Anybody can recommend a web page for testing by Kilotest. If the page has already been tested, you can recommend a retest. If not, you can recommend that a new page be tested.

A Kilotest manager can get a web page tested by approving a recommendation. Approval puts a testing job for the page into the queue. Jobs are drawn from the queue and performed. The test results are shown whenever somebody follows the link, Which web pages have been tested?, or the link, Which issues have been reported?.
