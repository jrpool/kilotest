Kilotest: Test recommendations

How recommendations work

Anybody can recommend a web page for testing by Kilotest. Recommendations that have been made and not yet approved are listed below.

A Kilotest manager can get a web page tested by approving a recommendation. A manager who wants a page tested that has not yet been recommended can recommend it and then approve the recommendation. Approval puts a testing job for the page into the queue. Jobs are drawn from the queue and performed. The test results are shown whenever somebody follows the link, Which web pages have been tested?
